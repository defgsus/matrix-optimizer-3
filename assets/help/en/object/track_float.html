The track can be used to group sequences together and arrange them in time. This object is mainly used for production purposes.

Currently the track object is not supported in the GUI because of developmental changes. It should work but it's not tested..

A track, when asked for a value, will return the value of the sequence that is at the given time on that track. If there is no sequence at that time, 0.0 is returned. So instead of connecting sequences for controlling something, you connect the track and place sequences on it.
