Equations

Equations are composed of statements and each statement can contain constants, variables, functions and operators.

statements

In the simplest form, a statement is a mathematical term leading to some result, for example:

5 + 3

or

sin(x * TWO_PI)

These examples are enough to produce a result. More statements can be entered separated by semicolons.

foo = 5 + 3; sin(foo)

In this example, a temporary variable foo is created and assigned. Then the sine of the variable is returned. All equations need exactly one statement that produces a result.

constants

Constants are names for fixed values. The compiler replaces the name with it's actual value.

Common built-in constants are:

!CONSTANTS!

variables

Variables represent values that can change for each execution of the equation.

functions

Functions usually take some input variables and return a result from some calculation.

Default built-in functions are:

!FUNCTIONS!
